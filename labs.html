Incident Response & Exploitation – H4cked (TryHackMe)

Overview: Investigated a compromised Linux machine by analyzing captured network traffic, then reproduced the attack to validate findings and retrieve the system flag.

Execution: Examined Wireshark packet captures to identify suspicious login attempts and attacker behavior. Determined that weak credentials were used for SSH access. Reperformed the attack using Hydra to brute-force the password, successfully gained shell access, and located the flag within the system.

Tools Used: Wireshark, Hydra, SSH, Linux CLI (grep, cat, find)

  • Identifying brute-force attempts through packet analysis
  • Using Hydra for credential attacks against SSH
  • Gaining and stabilizing shell access
  • Post-exploitation enumeration to locate sensitive files
