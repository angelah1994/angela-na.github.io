Lab Challenges

Hands-on cybersecurity challenges demonstrating practical skills in various security domains.

Web Application SQL Injection (Web Security)

Problem Statement: A vulnerable web application with a login form was susceptible to SQL injection. The task was to bypass authentication and retrieve sensitive data.

Approach: Tested manually with payloads, confirmed vulnerability, then automated extraction with SQLmap. Retrieved user data, cracked a password hash with John the Ripper, and used credentials to access a restricted host.

Tools Used: Burp Suite, SQLmap, JohnTheRipper, Browser DevTools

Key Lessons:

  • Importance of parameterized queries in web applications
  • Manual and automated SQL injection testing
  • Secure coding practices to mitigate SQLi

Web Server Directory Enumeration

Problem Statement: A misconfigured web server allowed directory listing and exposure of sensitive files.

Approach: Conducted reconnaissance with Gobuster and manual URL manipulation to discover open directories. Located and retrieved a hidden HTML file containing a flag.

Tools Used: Gobuster, Dirb, Browser

Key Lessons:

  • Risks of directory indexing
  • How to prevent accidental data exposure
  • Importance of secure web server configuration

SMB Share Enumeration

Problem Statement: Identified unsecured SMB shares that allowed anonymous access to sensitive files.

Approach: Used Nmap to scan for open SMB services, enumerated shares with smbclient, and accessed publicly available files to retrieve a flag.

Tools Used: Nmap, smbclient

Key Lessons:

  • Risks of leaving SMB shares open to anonymous users
  • Proper access controls for network shares
  • Need for regular service audits

PCAP File Analysis

Problem Statement: Traffic capture revealed sensitive data transmitted in plaintext over HTTP.

Approach: Loaded PCAP file in Wireshark, followed HTTP streams to recover directory paths and files, and extracted flag information directly from the capture.

Tools Used: Wireshark, Tshark

Key Lessons:

  • Dangers of transmitting data without encryption
  • How to analyze and reconstruct data from packet captures
  • Need to enforce HTTPS and secure transport protocols

Brute Force & Password Cracking

Problem Statement: Faced a weakly protected login portal requiring password discovery.

Approach: Captured login requests, then used Hydra and JohnTheRipper to brute force weak credentials.

Tools Used: Hydra, JohnTheRipper, Burp Suite

Key Lessons:

  • Risks of weak password policies
  • Rate limiting and lockout mechanisms
  • Strong password management strategies

Leviathan CTF (Capture The Flag)

Problem Statement: Participated in the Leviathan CTF game focusing on web, crypto, and forensics challenges to capture flags and score points.

Approach: Solved progressive tasks across categories — started with reconnaissance, moved to exploit development, and finished with forensic artifact recovery. Documented each flag and reproduction steps.

Tools Used: Burp Suite, Ghidra, binwalk, strings, Python (for automation), Volatility

Key Lessons:

  • Combining multiple security disciplines to solve complex CTF tasks
  • Systematic recon and note-taking speeds up teamwork
  • How to safely extract and validate forensic artifacts

Bandit Wargame (OverTheWire)

Problem Statement: Completed levels from the Bandit wargame to build practical Linux command-line and file-forensics skills.

Approach: Progressed through successive SSH levels on the Bandit server, using command-line tools to read files, decode encodings, navigate directories with non-standard permissions, and extract hidden data.

Tools Used: SSH, bash utilities (cat, ls, find), strings, hexdump, base64, tar, gzip, grep

Key Lessons:

  • Proficiency with Linux shell commands and file permissions
  • Techniques for decoding and extracting data from various encodings and archives
  • Importance of methodical problem solving and documenting commands used per level
